Technical Skills Highlights

Software Engineering & Cloud Computing

Data science models only add value when deployed to and utilised by end users. As such, software engineering and cloud computing are necessary in providing the application and platform in which these models can be integrated and deployed.

Data Science & Artificial Intelligence

As more data is made available, the more opportunities there are to make use of these to solve problems and increase efficiency in different domain areas. With deep learning being a rapidly developing field, it is important to keep up to date with latest developments, especially in application areas such as computer vision and natural language processing.

DevOps & MLOps

Continuous integration and continuous deployment pipelines are essential to ensure that the latest developed solutions are always made available to end users while the various parts of the applications are being continuously improved upon by the development team. With the advent of machine learning, a similar concept is applied to the end-to-end data solution pipeline, ensuring collaboration between data scientists and deployed model availability.

Biomedical & Other Domain Specific Areas

Domain specific knowledge is important in the application of data science techniques. Being able to quickly acquire working knowledge on the domain of interest allows for better problem formulation, data collection, processing and modelling.

MLOps: End-to-end CI/CD Pipeline For Machine Learning Models Using Github Actions, Docker and Azure Cloud Deployment Services

Github Actions is increasingly gaining popularity as the CI/CD pipeline tool of choice due to its YAML workflow file which makes setting up a pipeline simple and quick. In this project, I explored the use of Github Actions in setting up a simple MLOps pipeline that integrates a machine learning model with a web application for deployment upon model improvement.

Kaggle: Novozymes Enzyme Stability Prediction - Identify The Thermostable Mutations In Enzymes

This is an ongoing competition by Novozymes held on Kaggle from 21 Sep to 03 Jan that I am currently participating in. The aim of the competition is to come up with a model that predicts the melting point of proteins after they have undergone a single amino acid mutation or deletion. Being able to predict the thermostability of proteins will allow for cheaper and more efficent research that enables advancement in enzyme engineering which will contribute to sustainability and carbon neutrality.

Drug Discovery: Protein-ligand Interaction Prediction Using Deep Learning

The discovery of novel drugs remain a time consuming and costly process. Protein-ligand interaction is at the core of this process, which includes the identification, selection and design of molecules based on disease-specific target proteins. By using computational methods such as deep learning, the candidate molecule search space can be narrowed, leading to reduction in the amount of cost, labour and time required.
